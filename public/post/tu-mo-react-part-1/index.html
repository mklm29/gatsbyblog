Tự mò React: part 1

‘Học, học nữa, học mãi’, đuổi nghỉ!

React giờ thịnh hành toàn cầu toàn thiên hà, toàn vũ trụ, là Miss Universe của frontend, cho nên tuy công việc hiện tại chỉ sử dụng WP là chính, tui vẫn muốn tìm hiểu React là cái gì mà nhà nhà rì ắc, người người rì ắc.

Theo vốn từ vựng của tui thì React có nghĩa là re-act, tức là diễn lại. Yes! Thay vì viết web theo kiểu phân chia rõ ràng HTML,CSS,JS thì công ty của ông tỷ phú nọ muốn gom lại hết thành 1 file mà thoy! Thế là họ đẻ ra React! Oe oe oe

Quá trình tìm hiểu khá khó khăn vì framework này sử dụng định dạng rất khác biệt là JSX. Hoy nói nhiều thành nói dai, vô đề lun he

Một, hai, ba Dzôoooo

Cài đặt môi trường để sử dụng ‘đứa con của ông tỷ phú’ này theo giang hồ là rất đơn giản với create-react-app, nhưng do tui dốt, hay do cái mac của tui quá cùi mà Node chạy tám hướng. 🙁

Bước 1: tạo môi trường bằng create-react-app

One: cài đặt Nó-đè vô máy sẵn nha, không có N nó không có chạy đâu! :*
Two: cài create-react-app bằng terminal với cú pháp: npm install -g create-react-app
Three: Finish!!!! 😀

Giỡn hoy cài xong 2 bước trên bạn tiếp tục dùng terminal tạo thơ mục dự án:
$ mkdir studyreact
$ cd studyreact
$ npm init -y <- dòng này để tạo file package.json

Ở trên mới chỉ chuẩn bị folder, môi trường linh tinh, giờ mới bắt đầu vô đề:
$ npx create-react-app practice
$ cd practice
Thơ mục chứa dự án tui đặt tên là practice.

Lúc này bạn có thể tự tin gõ
$ npm start
để bắt đầu chạy web React.
Ô kìa browser mở 1 tab mới có url là localhost:3000, trang web trông như này với logo React xoay mòng mòng như cái bánh xe!

https://i.gyazo.com/b28c8d122356e438f9fa00eec3cf5b5b.png

Bước 2: điều hướng trang con

Cô gái này thật thú zị!
Đây là phần tui thấy ngộ nghĩnh và nhiều cảm xúc nhất trên con đường tìm hiểu framework này.

Điều hướng tiếng anh là Route, là phần hiển nhiên phải có như nấu ăn thì thêm muối vậy. Nhưng không, họ không tích hợp sẵn route vô React, bạn phải cài React Router để điều hướng qua các trang con trong web. Bất chợt tui thấy con đường làm quen ‘đứa con của tỷ phú’ này tối đen như tiền đồ của chị Dậu! Éc éc

Cú pháp để cài router như sau:
$ npm install react-router-dom

Cài xong bạn lại chạy npm start.

Tiếp theo chúng ta sẽ khai báo route. Ở bước này mình tách ra một file JS riêng rồi import lại vào app.js.
Một: tạo file routes.js trong directory /src/
Hai: import react và react-router vô file routes.js

https://i.gyazo.com/780fb4cb30b69950b106ca3c3fb1a501.png

Ba: tạo class mới và khai báo các đường dẫn

https://i.gyazo.com/8cde8de75133edc0c2d5d5661936aaf8.png

Bốn: import class mới tạo vô App.js
> import AppRouter from ‘./routes’;

Bởi vì bản chất của React là build component nên bạn sẽ thấy kiểu export rồi import qua lại rất nhộn nhịp.

Phần router này có 1 điểm cần lưu ý như sau: thông thường để khai báo điều hướng tới 1 trang nào đó bạn cần viết

<Route path=”/about/” component={About}>

Nhưng nếu muốn dẫn tới trang index/top/home, nói chung là trang đầu tiên của web thì cần thêm từ “exact” để router hiển thị component đúng người đúng chỗ. Thành ra nó sẽ như thế này:

<Route path=”/” exact component={Index}>

Phần này tui không rõ react-router họ viết như thế nào, nhưng nếu bạn đã từng sờ tới location.pathname của Javascript thì có lẽ sẽ hiểu. Ở MDN của Mozilla họ định nghĩa như vầy
